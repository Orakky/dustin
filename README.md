# dustin
A self-developed AI framework based on the Solon framework in Java.

## description
A self-developed AI framework based on the Solon framework in Java which provides a set of tools and components to help developers build ai chat applications more easily.

* support java 8 and above
* support solon 3.7.3 and above


## features

* easy to use
* persistent chat session
* support custom agent
* and more features are coming soon...
